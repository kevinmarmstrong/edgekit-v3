Add an agent to your web app without adding an AI backend.

edgekit is a drop-in sidecar for product workflows. Register your app's existing functions as tools, run the model in the browser when available, and require approval before the agent changes important state.

How it works

The agent uses your app. It does not replace it.

edgekit gives the model a controlled surface area: tools you register, state your app owns, and approvals your users can see.

01

Mount the sidecar

Add the edge-chat web component or use `createAgent()` when you need a custom interface.

02

Expose real actions

Wrap existing app functions as AI SDK tools with typed inputs and outputs.

03

Approve changes

Mark irreversible tools as guarded so the user approves before state changes.
